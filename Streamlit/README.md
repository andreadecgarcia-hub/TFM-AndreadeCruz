Aplicación web (carpeta Streamlit/)
1) Crea un Space nuevo → **SDK: Streamlit**.
   
2) Sube los ficheros de la app:
   - `app.py`
   - `detector.py`
   - `requirements.txt`
     
3) Define la clave en **Settings → Repository secrets**:
   - `OPENAI_API_KEY = sk-...`
   - (opcional) `OPENAI_MODEL = gpt-4o-mini`
